# Boundary-Crossing Leadership Survey (Streamlit)

A 20-item self-assessment grounded in **Akkerman & Bakkers (2011)** framework:
Identification 路 Coordination 路 Reflection 路 Transformation.

It calculates mechanism scores, plots a radar chart, and maps you to one of six
**Boundary-Crossing Leadership Archetypes**:
 Boundary Mapper 路  Bridge Architect 路  Reflective Sense-Maker 路  Ecosystem Catalyst 路 Л System Navigator 路  Integrative Weaver

##  How to Run Locally
```bash
# 1. Create virtual environment
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate

# 2. Install packages
pip install -r requirements.txt

# 3. Run Streamlit app
streamlit run app.py
